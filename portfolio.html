    Project Three

    Driveway-Sealing Management System for Contractor David Smith(In Progress)


    Overview

    Developing a web-based management system to streamline driveway-sealing requests, quotes, work orders, and billing for clients, with a focus on negotiation and dispute resolution functionalities. This project incorporates user registration, quote requests, order tracking, and bill generation, while allowing for real-time negotiation between contractor and client.


    Key Features to be Implemented

    • User Registration and ID Generation

      Collects and secures client details with a unique client ID.

    • Quote and Negotiation Workflow:

      Clients can request quotes with specific details and images of the driveway. They can negotiate terms with the contractor, with an iterative quote-and-response system until an agreement is reached or negotiations are terminated.

    • Order and Billing Process:

      Once a quote is accepted, an order of work is generated, and upon completion, a bill is issued. Clients can either pay immediately or dispute the bill, initiating a negotiation process until it is resolved or escalated.


    Additional Database Queries and Insights


    Implementation of advanced SQL queries to gather insights and support contractor decision-making:

    1. Top Clients:

      Identifies clients with the highest number of completed orders.

    2. Difficult Clients:

      Highlights clients who sent multiple requests but never followed up.

    3. Monthly Agreed Quotes:

      Lists quotes agreed upon within a specified month.

    4. Prospective Clients:

      Identifies registered clients who haven't submitted any quote requests.

    5. Largest Driveway Orders:

      Lists locations of the largest driveways worked on.

    6. Overdue Bills:

      Tracks unpaid bills overdue by one week.

    7. Client Status Tracking:

      Classifies clients as "Good" or "Bad" based on their bill payment history.


    Tools and Technologies

    • Frontend:

      React

    • Backend:

      Node.js, Express.js, SQL Database (MySQL with XAMPP)

    • Database Design:

      Includes E-R diagram with constraints and CREATE TABLE statements with primary/foreign keys.
